# Real-Estate-Sales-And-Listings-Analysis
Enterprise real estate analytics dashboard built using Power BI. It delivers insights on listings, pricing trends, property types, and sales performance. Features interactive filters, KPI tracking, and regional comparisons to support data-driven business decisions.


# 🏢 Enterprise Real Estate Analysis Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard designed to analyze real estate market trends. It provides insights into listings, pricing, sales performance, and property distribution across different states.

The goal is to help stakeholders make data-driven decisions using clear visualizations and KPIs.

---

## 📊 Key Features
- 📈 Total Listings, Active Listings & Sales Volume KPIs
- 💰 Average Sale Price Analysis
- 📅 Monthly Sales & Listing Trends
- 🏠 Property Type Distribution (Condo, Single Family, etc.)
- 🌍 State-wise Performance Comparison
- 🧾 Brokerage Transaction Analysis
- 🎯 Interactive Filters (Price Range, Year, Property Type, State)

---

## 🛠️ Tools & Technologies
- Power BI
- SQL (Data Cleaning & Transformation)
- Excel / CSV (Dataset)
- DAX (Calculated Measures)
